# City Distance Analyzer

This project analyzes distances between cities in Central Europe.  
Given a selected city, the application finds the closest and furthest cities and visualizes all distances in a matrix.

## Features

- Search for a city by name
- Find closest and furthest cities
- Highlight selected, closest, and furthest cities in the UI
- Display a full distance matrix between all cities
- Dynamic DOM rendering using JavaScript

## How It Works

- A dataset of cities and distances is loaded
- The user inputs a city name
- The app calculates:
  - Closest city
  - Furthest city
- Results are displayed visually and in a table
